## Getting started

```bash
# clone
git clone https://github.com/jchornales/portfolio.git
cd portfolio

# install
npm install

# dev server (http://localhost:4321)
npm run dev
```

```bash
npm run build    # production build → dist/
npm run preview  # preview the build locally
npm run check    # TypeScript + Astro type check
```
